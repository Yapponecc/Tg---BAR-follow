# TG Live Overlay

Minimalist Telegram subscriber bar for TikTok LIVE Studio.

## Setup

```bash
npm install
```

Create `.env` from `.env.example` and fill:
- `TG_BOT_TOKEN`
- `TG_CHAT_ID`

Run:

```bash
npm start
```

Open:
- `http://127.0.0.1:8787/?goal=10000&poll=15000&label=telegram`
